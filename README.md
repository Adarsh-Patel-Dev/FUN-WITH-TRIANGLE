# FUN WITH TRIANGLE
 ### An app to learn about triangle. It consists of four part:
 * QUIZ ON TRIANGLES
 * WHETHER GIVEN ANGLES WILL FORM TRIANGLE?
 * CALCULATE AREA OF TRIANGLE
 * FIND HYPOTENUSE OF RIGHT ANGLED TRIANGLE
